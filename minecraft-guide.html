Minecraft Guide

Punishments

All Rules and Punishments can be found at RupeCraft.com/rules or by doing /rules in-game.

You should check the history of players to check for warnings and past punishments before punishing.

When punishing, you should add the Rule nb to the reason. (Ex. /tempban r7mexpert 2w Fly Hacks - Rule #1)

The following punishments can be applied to players, and should be done so through the following:

Check History Command: /history [user] - Helper+

Warn Command: /warn [user] [reason] - Helper+

Kick Command: /kick [user] [reason] - Helper+

TempMute Command: /tempmute [user] [duration] [reason] - Helper+

Mute Command: /mute [user] [reason] - Helper+

Unmute Command: /unmute [user] [reason] - Moderator+

TempBan Command: /tempban [user] [duration] [reason] - Helper+

Ban Command: /ban [user] [reason] - Helper+

Unban Command: /unban [user] - Moderator+

Temp-IP Command: /tempipban [user] [duration] [reason] - Admin+

IP Ban Command: /ipban [user] [reason] - Admin+

UnIPban Command: /unbanip [user] [reason] - Admin+

Change Reason Command: /change-reason [id] [new reason]

Restore Inventories

If a Inventory Restore is required (Due To Hackers, Corruption, ...) then admins should do this through /ir view [player] .
